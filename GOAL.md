# Goal

A simple, minimal book sharing application

# Functionalities

- User can create an account & login
- User can write a book
- User can comment on other people's books (threaded comments)
- User can share their book through a URL link with the world (like on social media etc)

# Requirements
- Utilizing the following services: Firebase Firestore, Firebase Storage, Firebase static hosting (not app hosting) & Firebase Functions
- Minimal and as simple of a codebase as possible
- Utilize the exact design language of writebook/

# Tech specs
- Frontend: React
